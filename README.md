# 🎬 Movie Analysis and Recommendation System

A Machine Learning based movie recommendation system that suggests similar movies using TF-IDF Vectorization and Cosine Similarity. The project includes an interactive Streamlit web application and Power BI dashboard for movie analytics and visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Power BI

## Features
- Personalized Movie Recommendations
- Content-Based Filtering
- Interactive Dashboard
- Data Visualization
- Movie Analytics
